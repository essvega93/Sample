## Getting Started

Please make sure that you already setup Flutter environement. 

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/essvega93/Sample.git
   ```
2. Install packages
   ```sh
   flutter pub get
   ```
3. To launch the app in the Simulator, ensure that the Simulator is running and enter:
   ```sh
   flutter run
   ```
